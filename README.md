Descrição
Gothic Data Art é uma aplicação de meteorologia que transforma dados climáticos reais em arte generativa. Ao selecionar uma cidade, os dados de temperatura, humidade, qualidade do ar e vento são visualizados num canvas com uma estética sombria e dramática, onde cada variável climática se traduz numa expressão visual única.
A aplicação combina data art com design gótico, tornando a informação meteorológica numa experiência visual imersiva.

Funcionalidades

Seleção de cidade para obter dados meteorológicos em tempo real
Visualização artística dos dados no canvas (temperatura, humidade, vento, qualidade do ar)
Estética gótica e sombria com elementos gerados proceduralmente
Atualização dinâmica da composição visual com base nos dados recebidos


Como utilizar

Abrir a aplicação no browser
Escolher uma cidade no seletor "✦ CHOOSE A CITY ✦"
Observar a composição visual gerada a partir dos dados meteorológicos dessa cidade


Tecnologias

HTML5 Canvas (arte generativa)
CSS3
JavaScript (vanilla)
API meteorológica externa (dados em tempo real)
